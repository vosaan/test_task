Вот ваше тестовое задание:

Страница No.1 - форма регистрации или форма логина (в этом случае
пришлите нам логин и пароль)
Страница No.2 - отображать погоду на сегодня по Запорожью, которая
парсится по этой ссылке - http://www.gismeteo.ua/city/daily/5093/
Страница No.3  - форма обратной связи.  Поля (имя, email, сообщение)
Страница No.4 - Отображение списка фидбеков, которые добавляются на
странице No.3.

Возможные плюсы при выполнении:
1. Кеширование данных о погоде.
2. Дополнительная javascript валидация полей для формы, кроме основной
php на стороне сервера.
3. Любой шаблонизатор типа smarty
4. Красивый и откоментированый код, и т..д.
